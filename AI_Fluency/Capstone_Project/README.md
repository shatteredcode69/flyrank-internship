# 🏆 AI Fluency Capstone: General AI Fluency Impact Project

This repository contains the deliverables for the General AI Fluency capstone project. The goal was to master the AI stack, build a personal brand with a real website, and ship a personal agent.

## 📌 Capstone Details
- Type: Capstone
- Track: General AI Fluency
- When: Week 6
- Workload: 12h
- Focus: Master the AI stack, build a personal brand with a real website, ship a personal agent

- Q&A: https://internship.flyrank.ai/intern/assignments/fl-cap?tab=qa
- Feedback: https://internship.flyrank.ai/intern/assignments/fl-cap?tab=feedback

## 🏗️ Deliverables

### 1. Personal Brand Website
A polished one-page portfolio website was built in [index.html](index.html) to present the capstone project professionally.

Highlights include:
- a refined dark visual theme
- clear sections for the capstone mission, skills, project highlights, and learning journey
- links to GitHub profile and repositories
- an emphasis on AI, cloud, DevOps, and backend growth

### 2. Personal AI Agent
A structured system prompt and guidance were created in [agent-prompt.md](agent-prompt.md) to help represent the student professionally in interviews, portfolio reviews, and recruiter conversations.

The agent is designed to:
- answer questions clearly and accurately
- stay within the provided knowledge base
- maintain a professional and confident tone

## 📂 Project Highlights
- AI Fluency capstone storytelling
- Cloud and DevOps learning path
- Portfolio presentation with repository evidence
- Personal agent framework for professional representation
